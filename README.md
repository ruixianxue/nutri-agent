NutriAgent AI is an open-source, privacy-focused web application that helps you make safer food choices. By combining computer vision, real-time food databases, and Google's Gemini 2.5 AI, it analyzes ingredients against your specific health profile (like Diabetes, Hypertension, or Allergies).

## üèÅ How to Run Locally

You can run this project on your own computer in 5 minutes.

### Prerequisites
1.  [Node.js](https://nodejs.org/) (Version 18+)
2.  A **Google Gemini API Key**. It is completely free. [Get one here](https://aistudio.google.com/app/apikey).

### Steps

1.  **Clone this repository**
    ```bash
    git clone https://github.com/ruixianxue/nutri-agent.git
    cd nutri-agent
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the app**
    ```bash
    npm run dev
    ```

4.  **Open in Browser**
    Click the link shown in your terminal (usually `http://localhost:5173`).

5.  **Enter your Key**
    Click the **Key Icon** in the top-right corner and paste your Gemini API Key.

## üåç How to Deploy (Free)

3.  Import your `nutri-agent` repository.
